### Билет 3. Уравнение экспоненциального процесса. Свойства его решения. Время удвоения и уменьшения вдвое. Конкретные примеры экспоненциальных процессов.

Экспоненциальные процессы описываются простым дифференциальным уравнением:

$$\frac{dx}{dt} = \lambda x.$$

Решение этого уравнения при начальном условии $x(0) = x_0$ имеет вид:

$$x(t) = x_0 e^{\lambda t}.$$

**Свойства решения:**
1. При $\lambda > 0$ процесс является экспоненциально возрастающим (неограниченный рост).
2. При  $\lambda < 0$ процесс является экспоненциально убывающим.
3. Решение обладает свойством постоянства времени, за которое значение переменной изменяется в определенное число раз. Спустя время **τ** (период удвоения/уменьшения вдвое), переменная становится равной $$x(t + \tau) = 2x(t)$$ (для $\lambda > 0$) или $$x(t + \tau) = x(t)/2$$(для $\lambda < 0$). Это время вычисляется как $$\tau = \frac{\ln 2}{|\lambda|}$$

**Примеры экспоненциальных процессов:**

1. **Радиоактивный распад.** Процесс убывания числа нераспавшихся ядер, где $\lambda < 0$ — постоянная распада. Количество вещества: $$x(t) = x_0 e^{-\frac{\ln 2}{\tau} t}$$где $\tau$ — период полураспада.
2. **Неограниченное размножение популяции (закон Мальтуса).** Скорость роста пропорциональна текущей численности: $dN/dt = \lambda N$. Рост идет по экспоненте $$N(t) = N_0 e^{\lambda t}$$, где $\lambda = \alpha - \beta$ — удельный прирост (рождаемость минус смертность).
3. **Охлаждение нагретого тела.** Скорость охлаждения пропорциональна разнице температур между телом и средой: $\dot{T} = k(T - \theta)$. Решение имеет вид  $$T(t) = \theta + (T_0 - \theta)e^{kt}$$, где  $k < 0$, что соответствует экспоненциальному приближению температуры тела к температуре окружающей среды.
4. **Разряд конденсатора** и другие процессы, где скорость изменения величины пропорциональна самой величине.
